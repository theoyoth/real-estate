# Real estate app

## Explain

I'm using API from rapid API, you can find real estate that is sold or rented in this web

## Getting Started

After you clone this repo, open terminal and type 'npm install' and the run it.
then, run the development server.

```bash
npm run dev
# or
yarn dev
```
